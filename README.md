
# moz-addon-sdk-installer

Easily install and update Mozilla Addon SDK Installer


## installation:

```bash
⇒ wget http://git.io/vvEpZ | bash
```


## usage:

```bash
⇒ moz-sdk
```

After a successfull install/update, the addon-sdk will be available with the command `cfx`.
